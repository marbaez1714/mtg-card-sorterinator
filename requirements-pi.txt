# Run these once on the Raspberry Pi (not via pip):
#
#   sudo apt update
#   sudo apt install -y python3-picamera2 python3-libcamera
#
# picamera2 on PyPI is a stub — the real library ships with Raspberry Pi OS
# Camera Module 3 uses this same picamera2 + libcamera stack (no separate "picamera3" package).
# and links against system libcamera. Do not pip install it.
#
# After the apt install, also install the pip packages:
#   pip install -r requirements.txt
#
# Optional 128x64 OLED (see oled.py, AGENT.md):
#   I2C: enable I2C in raspi-config; OLED_ENABLED=1 (default OLED_INTERFACE=i2c).
#   SPI: enable SPI in raspi-config; OLED_ENABLED=1 OLED_INTERFACE=spi
#        (wire per AGENT.md; OLED_DRIVER=ssd1306 or sh1106 if needed).
